====================================================
  新疆WebGIS监测与分析系统
  Vue3 + Element Plus 前端项目
====================================================

项目位置: E:\webgis\xinjiangweb

一、项目介绍
----------------------------------------------------
这是一个基于Vue3和Element Plus的现代化WebGIS应用，
用于农作物监测与分析。

主要功能模块:
✓ 监测主控台 - 数据总览和KPI展示
✓ 影像数据管理 - 上传、浏览、筛选影像
✓ 分类分析任务 - 创建和管理分析任务
✓ 结果查看与比对 - 地图展示和差异分析
✓ 图表报表 - 数据可视化和报告生成

二、技术特点
----------------------------------------------------
✓ Vue 3 Composition API - 现代化开发体验
✓ Element Plus - 美观的UI组件库
✓ ECharts - 强大的图表功能
✓ 响应式设计 - 支持各种屏幕尺寸
✓ 模块化架构 - 易于维护和扩展

三、快速启动
----------------------------------------------------
1. 安装依赖:
   npm install

2. 启动开发服务器:
   npm run dev

3. 打开浏览器:
   http://localhost:3000

四、项目结构
----------------------------------------------------

五、界面预览
----------------------------------------------------
1. 监测主控台 (/)

2. 影像管理 (/image-management)

3. 任务管理 (/task-management)

4. 结果比对 (/result-compare)

   - 图层控制
   - 差异列表

5. 图表报表 (/report)
   - 多维度图表
   - 数据表格
   - 报告生成
